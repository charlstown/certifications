## Con Azure Locks, ¿Qué niveles de bloqueo están disponibles? (Escoja dos)

- [x] CanNotDelete
- [x] ReadOnly
- [ ] Modify
- [ ] FullControl

> CanNotDelete: No se puede eliminar un recurso. ReadOnly : Un recurso puede leerse pero no borrarse o actualizarse.
